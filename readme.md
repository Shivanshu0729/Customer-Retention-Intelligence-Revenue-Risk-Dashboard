Customer Retention – Revenue Risk Analytics Dashboard

An end-to-end Machine Learning + Business Intelligence project that predicts customer churn and quantifies its financial impact through an interactive dashboard.

📌 Overview

Customer churn is one of the biggest challenges businesses face. Losing customers directly impacts revenue and growth.

This project provides a data-driven solution by combining:

Churn prediction using Machine Learning
Customer segmentation and risk analysis
Revenue impact and ROI estimation
Interactive dashboard for business insights

🎯 Problem Statement
Businesses often struggle to:

Identify customers likely to churn
Understand key factors behind churn
Estimate revenue loss due to churn
Take proactive actions to retain customers

💡 Solution
This project builds a Customer Retention Intelligence System that:

Predicts customer churn using ML
Segments customers based on value & engagement
Identifies high-risk customers
Calculates Customer Lifetime Value (CLV)
Estimates revenue loss and retention ROI
Provides actionable business insights

🧠 Key Features

✔️ Churn Prediction using XGBoost
✔️ Customer Segmentation (Value, Engagement, Tenure)
✔️ Risk Classification (Critical, High, Medium, Low)
✔️ Financial Impact Analysis (CLV, Revenue Loss)
✔️ ROI Calculator for Retention Strategies
✔️ Interactive Dashboard with Filters
✔️ Downloadable Reports (CSV)
